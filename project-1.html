TECHNICAL ARCHITECTURE

System Design

Every layer engineered for industrial reliability — from the RS485 bus to the LoRa uplink.

📡

RAK3172 LoRaWAN

Long-range, low-power wireless module providing reliable uplinks over large factory floors and remote sites where Wi-Fi is infeasible.

Class A / Class C
🔌

RS485 / Modbus RTU

Industry-standard half-duplex differential serial bus. Immune to common-mode noise, supports 32+ nodes on a single bus segment up to 1200m.

115200 baud / CRC16
⚡

ESP32 Core

Dual-core 240MHz Xtensa LX6 MCU handling Modbus polling, data aggregation, UART-to-LoRa bridging, and local buffering in parallel.

FreeRTOS multi-task
🛡️

EMI Hardened PCB

Custom 4-layer PCB with dedicated ground plane, transient voltage suppressors (TVS), and 120Ω termination resistors for signal integrity.

IEC 61000-4 class
💾

Offline Buffer

SPIFFS-based circular log on internal flash retains 72h of readings. Automatic replay on reconnection prevents data loss during outages.

4MB NVS partition
☁️

Cloud Telemetry

Decoded payloads forwarded via network server to MQTT broker. Compatible with Node-RED, InfluxDB, and Grafana dashboards.

Cayenne LPP payload

PROJECT STATUS

Development Timeline

From concept validation to active field deployment.

PHASE 01 · COMPLETED
Requirements & Site Audit
Professional site audits of laboratory and industrial equipment to map Modbus device catalog, wiring topology, and communication constraints.
PHASE 02 · COMPLETED
Hardware Prototype & PCB Design
Custom 4-layer PCB designed with RS485 transceiver, isolation stage, TVS protection, and RAK3172 footprint. First revision fabricated and tested.
PHASE 03 · COMPLETED
Firmware Development
FreeRTOS-based firmware with Modbus RTU polling engine, LoRaWAN OTAA join, Cayenne LPP payload encoding, and SPIFFS offline buffer.
PHASE 04 · IN PROGRESS
Field Deployment & Validation
Active deployment at industrial sites. Collecting long-duration reliability data, refining alarm thresholds, and iterating on enclosure cable routing.
PHASE 05 · PLANNED
V2 Hardware Revision
Planned: onboard 4G fallback modem, local e-ink status display, enhanced galvanic isolation, and DIN rail spring terminal connectors.
